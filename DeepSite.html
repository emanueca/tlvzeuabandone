Desenvolvimento Web

Prof. Ana Oliveira

10 vagas restantes Inscrições abertas

Este projeto tem como objetivo ensinar os fundamentos do desenvolvimento web moderno, incluindo HTML5, CSS3 e JavaScript ES6+. Os participantes terão a oportunidade de desenvolver um projeto prático ao longo do curso.

Objetivos:

  • Compreender a estrutura básica de páginas web
  • Aprender a estilizar páginas com CSS moderno
  • Implementar interatividade com JavaScript
  • Desenvolver um projeto prático completo

Cronograma:

Módulo 1: Introdução ao HTML5

15/08 - 19/08

Módulo 2: CSS Moderno

22/08 - 26/08

Módulo 3: JavaScript ES6+

29/08 - 02/09

Projeto Final

05/09 - 16/09

Requisitos:

Conhecimento básico de informática. Não é necessário experiência prévia em programação.
